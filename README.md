# Revivar Software Engineering Internship Assessment

A responsive web app that lets users design and download a personalized “Thank You” card using images from the Unsplash API.

## Features

- Fetches 4 random images from Unsplash for card design.
- Users can:
  - Pick an image.
  - Enter their name to personalize the card.
  - Generate a custom card with “Thank You” text on top and their name at the bottom.
  - Download the generated card as an image.
- Image search with pagination.
- Font type and colour selection

## Tech Stack

- **Frontend**: React (with TypeScript)
- **Styling**: TailwindCSS
- **Data Fetching**: Unsplash API + TanStack Query
- **Image Export**: html-to-image
